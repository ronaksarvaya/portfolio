## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Navigate to the project directory:
```bash
cd portfolio-nextjs
```

2. Install dependencies:
```bash
npm install
```

3. Copy images to the public folder:
```bash
# Copy the following images from the parent directory to portfolio-nextjs/public/:
# - ronakimg.png
# - spidy.png
# - Project1.png
# - Project2.png
# - Project3.png
# - contactimg.png
```

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
npm start
```
